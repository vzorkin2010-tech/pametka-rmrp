ФЗоП 11.1 Права полиции
  • Требовать прекращения противоправных действий
  • Проверка паспорта и документов
  • Останавливать ТС для проверки документов
  • Требовать покинуть место преступления
  • Требовать разойтись при нарушении порядка
ПК 20.2 Обыск ТС
  • а) В зоне ЧП | ВП | КТО
  • б) В случае нарушения правил
  • в) Наличие ТС в зоне преступления
  • е) Проезд на охраняемую территорию
1. Адвокат несёт только рекомендательный характер
2. К ОМОНУ/СПЕЦНАЗУ/ГРОМу/ФСБ А обращаемся только по послед. 2 цифрам id
3. ФЗоГСЗОтСтатья 15. Территории органов МВД
4. Озвучиваем права задержанного только по запросу
ПК 11.1 Допуск на место проц. действий
  • а) Сотрудник, проводящий действия
  • б) Задерживаемый
  • в) Один адвокат
  • г) Не более 1 предст. прокуратуры
  • д) Не более 1 предст. СК России
  • ж) Предст. руководства организации
  • з) Не более 1 предст. УСБ ФСБ
  • и) Сотрудник, передавший задержание
Процессуальный Кодекс
7.1 Основания для задержания
  • а) Лицо застигнуто при совершении или после
  • б) Очевидцы (3+ чел) укажут на лицо
  • в) На лице или одежде явные следы
  • г) Для установления личности (если скрывает)
  • ж) Наличие лица в розыске
  • н) Ориентировка на лицо/ТС
8.1 Права задержанного
  • а) Не свидетельствовать против себя
  • б) Требовать предоставления адвоката
  • в) Требовать присутствия адвоката (гос.)
  • г) Консультация с адвокатом (до 5 мин)
  • д) Один тел. звонок (до 2 мин)
10.1 Порядок задержания
  • а) Применить спец. средства (наручники)
  • б) Представиться, показать жетон
  • в) Огласить статьи нарушения
  • г) Произвести личный обыск
  • д) Доставить в участок
18.2 Порядок ареста
  • а) Уведомить о начале стадии ареста, разъяснить причину
  • б) Личный обыск с конфискацией нелегала
  • в) Произвести арест лица
14.1 Основания для окончания задержания
  • а) Недостаточно доказательств
  • б) Не предусмотрено лишения свободы
  • в) Истек срок (60 мин)
